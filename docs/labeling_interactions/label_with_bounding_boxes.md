# Labeling with Bounding Boxes

You can record your observations by interacting with bounding boxes in addition to using [keyboard shortcuts]({% link labeling_interactions/label_with_hotkeys.md %}). 

For this, you should either import a tracking file or draw bounding boxes around the subjects of interest.
