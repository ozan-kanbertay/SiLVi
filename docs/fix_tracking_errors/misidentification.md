# Fixing Misidentification

To quickly correct the name of an individual who was assigned a wrong name by the model, right-click on a bounding box and select the correct name from the drop-down menu.

<video src="../assets/videos/label_individual_via_right_click.mp4" class="help-video" muted loop autoplay></video>

Alternatively, you can click on a bounding box. A menu with the options for changing the track properties will open. Then, make sure <strong>Change name</strong> option is chosen, select the correct name from the drop-down menu, and click <strong>Save</strong>.

<video src="../assets/videos/label_individual_via_bounding_box.mp4" class="help-video" muted loop autoplay></video>
