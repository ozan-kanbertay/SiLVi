## Version 0.5.0
### New Features
- Assigning names and colors to species
- Flexible tracking box line width and opacity
- Importing example files with a single button
- Resizable tracking boxes
- Draw tracking boxes around subject/objects

### Improvements
- Clickable table of contents and snippets to help menu
- Asking for user confirmation before quitting the app
- Immediate visual feedback over buttons
- Visual indicators for enabled modes
- Shortcuts to enable/disable zoom and drawing modes

### Bug Fixes
- Failure for importing behavior files
- Disappearing button for seeking to a specific frame in the main video
- Unintended update of the zoom canvas on mouse click
- Importing a CSV file with semicolon as the separator
- Pressing Enter jumps to the given frame

## Version 0.4.0
### New Features
- Added zoom functionality. Hold down your mouse over the main view and draw a rectangle to zoom in on a specific area. Release the mouse to view the zoomed area in a new window, where you can adjust the zoom scale or use the settings menu. The window for zoomed area can be moved and hidden freely. 

### Improvements
- The mouse pointer changes to a cursor over the tracks, indicating that they are clickable.
